# 📊 E-Commerce Sales Analytics & Forecasting Dashboard

An interactive Business Intelligence dashboard developed using **Power BI, Power Query, DAX, Python, and Prophet** to analyze Amazon e-commerce sales data and forecast future business performance.

---

## 📌 Project Overview

This project analyzes **128,975 Amazon e-commerce transactions** collected between **March 2022 and June 2022**. The dashboard transforms raw sales data into meaningful business insights through data preprocessing, interactive visualizations, KPI tracking, and machine learning-based forecasting.

---

## 🎯 Objectives

- Analyze overall sales performance
- Track Revenue, Orders, Units Sold, and Cancellation Rate
- Compare regional and category-wise performance
- Forecast future sales using Prophet
- Generate business insights for decision-making

---

## 📂 Dataset Information

- **Dataset:** Amazon Sale Report
- **Records:** 128,975
- **Columns:** 24
- **Time Period:** March 2022 – June 2022

---

## 🛠 Data Preprocessing

The dataset was prepared using Power Query by performing:

- Date format correction
- Missing value handling using **Group By + Median Imputation**
- Data type correction
- Feature engineering
- Region grouping
- Order Value Segmentation
- Top-N filtering
- Data model optimization

---

## 📊 Dashboard Pages

### Executive Overview
Provides a summary of overall business performance using key KPIs and sales trends.

### Revenue Performance
Analyzes revenue across regions, categories, and customer order value segments.

### Orders Analysis
Tracks order distribution, regional performance, and monthly order trends.

### Units Analysis
Evaluates units sold across different categories and regions.

### Cancellation Analysis
Monitors cancellation trends and operational performance.

### Predictive Analysis
Implements **Prophet** forecasting to predict Revenue, Orders, and Units Sold for the next month.

### Business Insights
Summarizes key findings and recommendations to support business decision-making.
